PocketAgent v6 Install

Your AI agent with a real Linux computer on your phone. Two-minute setup.

Step 1 · 1 min
Install Termux
Open F-Droid → Termux on your phone and tap Install.

Don't use the Play Store version — it's deprecated and won't work.
Step 2 · 30 sec
Run the installer in Termux
Open Termux, then tap the button below to copy the command. Paste it into Termux (long-press anywhere in Termux → Paste) and press Enter.
curl -sL https://tinyurl.com/266a3atb | bash
Step 2 alt
Or scan this QR code in Termux
Some keyboards can scan QR codes directly. Or use any QR scanner app to get the command.
QR encodes the full install command
Step 3
Copy the token
The installer prints a token at the end (looks like eZhDtZXAlfskmRDi...). Copy it — you'll paste it into the PocketAgent app.
Step 4
Install PocketAgent APK
Download the latest APK from GitHub Releases, install it, open it, go to Settings → Termux → paste the token → Save → Reconnect.
Done
Start chatting
The AI now has full access to your Termux. Ask it anything — build a website, install python, write a script, anything you can do in Termux.
